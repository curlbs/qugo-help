# Отчёт по балансу

Отчёт по балансу — это полная выписка о движении денежных средств на вашем счёте в Qugo.

## Какая информация включена

- Пополнения баланса
- Выплаты исполнителям
- Списания комиссии сервиса
- Оплата НПД (налог на профессиональный доход) за исполнителей

## Как получить отчёт

1. Войдите в [личный кабинет](https://lk.qugo.ru).
2. Перейдите в раздел **Баланс** → **Отчёт**.
3. Выберите период.
4. Нажмите **Скачать отчёт**.

Отчёт формируется в формате XLSX и содержит детализацию по каждой операции.

:::info
Отчёт обновляется в реальном времени — вы всегда видите актуальное состояние баланса.
:::
