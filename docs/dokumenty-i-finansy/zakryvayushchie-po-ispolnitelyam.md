# Закрывающие документы по исполнителям

В любой момент вы можете скачать полный комплект закрывающих документов по работе с исполнителями.

## Какие документы доступны

- **Договоры** — с каждым исполнителем
- **Акты выполненных работ** — по каждому заданию
- **Чеки** — сформированные самозанятыми исполнителями
- **Приложения** — дополнительные документы к договорам
- **Опись документов** — сводный реестр

## Как скачать

1. Войдите в [личный кабинет](https://lk.qugo.ru).
2. Перейдите в раздел **Документы** → **По исполнителям**.
3. Выберите период и/или конкретных исполнителей.
4. Нажмите **Скачать**.

Документы формируются в виде ZIP-архива с папками по каждому исполнителю.

:::tip Совет
Скачивайте закрывающие документы ежемесячно и передавайте в бухгалтерию для своевременного учёта.
:::
