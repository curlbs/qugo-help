# Закрывающие документы с платформой

В личном кабинете доступен акт на комиссию сервиса Qugo. Также вы можете настроить обмен документами через систему ЭДО.

## Как получить акт на комиссию

1. Войдите в [личный кабинет](https://lk.qugo.ru).
2. Перейдите в раздел **Документы** → **С платформой**.
3. Выберите период.
4. Скачайте акт.

## Настройка ЭДО с Qugo

Для автоматического обмена закрывающими документами подключите ЭДО через одну из поддерживаемых систем:

- **СБИС**
- **Контур Диадок**

Реквизиты для поиска контрагента:
- Организация: ООО «КЬЮГО ТЕК»
- ИНН: 7727452605

:::info
После подключения ЭДО акты будут приходить автоматически в вашу учётную систему.
:::
