# Инструкции для исполнителей

Готовые инструкции, которые помогут вашим исполнителям быстрее зарегистрироваться на платформе и начать получать выплаты. Скачайте нужные и отправьте исполнителям.

## Доступные инструкции

### Регистрация на платформе
Пошаговая инструкция для самозанятых исполнителей: как скачать приложение, создать аккаунт и заполнить анкету.

### Регистрация для исполнителей из Кыргызстана
Отдельная инструкция для исполнителей-резидентов Кыргызстана с учётом особенностей регистрации.

### Как изменить банковские реквизиты
Инструкция по обновлению реквизитов для получения выплат: номер счёта, номер карты.

### Как принять задание
Пошаговое описание: как найти задание в приложении, ознакомиться с условиями и принять его.

### Как подписать акт
Инструкция по подписанию акта выполненных работ в мобильном приложении.

:::tip Совет
Отправьте инструкцию по регистрации вместе с приглашением — это значительно сократит количество вопросов от исполнителей.
:::
