# Приглашение с готовыми анкетами

Если вы уже сотрудничали с исполнителями и знаете их данные, загрузите анкеты заранее. Исполнителям останется только подтвердить корректность информации и сохранить анкету — регистрация пройдёт быстрее.

## Как загрузить анкеты

1. Перейдите в раздел **Исполнители** → вкладка **Реестр исполнителей**.
2. Скачайте шаблон файла с расширенными полями.
3. Заполните данные исполнителей: ФИО, телефон, ИНН, банковские реквизиты.
4. Загрузите файл на платформу.

## Что происходит после загрузки

- Исполнители получают SMS-приглашение
- При регистрации видят предзаполненную анкету
- Проверяют данные, вносят правки при необходимости
- Подтверждают и сохраняют

## Когда использовать

- Переход с другого сервиса, где уже есть база исполнителей
- Работа с постоянной командой, чьи данные у вас на руках
- Нужно максимально упростить регистрацию для исполнителей
