# Массовое приглашение к регистрации

Если нужно пригласить сразу много исполнителей — загрузите файл с их номерами телефонов. Одним файлом можно пригласить любое количество людей.

## Как пригласить массово

1. Перейдите в раздел **Исполнители** → вкладка **Избранные исполнители**.
2. Нажмите **Пригласить исполнителя** → **Реестром**.
   - Альтернативно: перейдите во вкладку **Реестр исполнителей**.
3. Скачайте шаблон файла.
4. Заполните шаблон номерами телефонов исполнителей.
5. Загрузите заполненный файл на платформу.

Все исполнители из файла получат SMS-приглашения.

## Требования к файлу

- Формат: XLSX или CSV
- Обязательное поле: номер телефона
- Номера в формате: +7XXXXXXXXXX

:::info
Для ещё более простой регистрации исполнителей используйте [приглашение с готовыми анкетами](s-gotovymi-anketami) — исполнителю останется только подтвердить данные.
:::
