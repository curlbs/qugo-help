# Приглашение исполнителей по ссылке

Если вы общаетесь с исполнителями в мессенджерах или соцсетях и у вас нет их номеров телефонов — отправьте ссылку-приглашение.

## Как создать ссылку

1. Перейдите в раздел **Исполнители** → вкладка **Избранные исполнители**.
2. Нажмите **Пригласить исполнителя**.
3. Выберите **Ссылкой**.
4. Скопируйте сгенерированную ссылку.
5. Отправьте её исполнителям любым удобным способом: в мессенджере, по email, в группе.

## Когда использовать

- Исполнители общаются с вами в Telegram, WhatsApp или других мессенджерах
- Нужно разместить ссылку в группе или чате
- У вас нет номеров телефонов исполнителей

:::tip Совет
Ссылку можно использовать многократно — каждый перешедший по ней исполнитель будет привязан к вашей компании после регистрации.
:::
