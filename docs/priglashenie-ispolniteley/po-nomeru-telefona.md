# Индивидуальное приглашение по номеру телефона

Самый простой способ пригласить исполнителя — отправить приглашение на его номер телефона. Подходит для тестирования платформы и работы с небольшим количеством исполнителей.

## Как пригласить

1. Перейдите в раздел **Исполнители** → вкладка **Избранные исполнители**.
2. Нажмите **Пригласить исполнителя**.
3. Выберите **По номеру телефона**.
4. Введите номер телефона исполнителя.
5. Нажмите **Отправить приглашение**.

Исполнитель получит SMS со ссылкой на регистрацию в приложении Qugo.

## Когда использовать

- Пробный запуск платформы с несколькими исполнителями
- Ежедневный приём новых исполнителей
- Работа с небольшой командой

:::info
Если у вас нет номеров телефонов исполнителей, используйте [приглашение по ссылке](po-ssylke).
:::
