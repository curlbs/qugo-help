# Оплата заданий с изменением стоимости (арбитраж)

Платформа позволяет изменить стоимость задания в момент оплаты. Это полезно, когда фактический объём работ отличается от запланированного.

## Как изменить стоимость при оплате

1. Перейдите в раздел **Задания** и найдите нужное задание.
2. Откройте задание и нажмите **Оплатить**.
3. В форме оплаты измените сумму на фактическую.
4. Укажите причину изменения (опционально).
5. Подтвердите оплату.

## Когда использовать арбитраж

- Исполнитель выполнил больший или меньший объём работ
- Изменились условия задания в процессе выполнения
- Нужно скорректировать оплату по договорённости с исполнителем

:::info
Исполнитель получит уведомление об изменённой сумме выплаты.
:::
