# Массовое тегирование исполнителей

Теги позволяют группировать исполнителей по любым признакам: город, тип работ, проект. Это упрощает отправку документов через ЭДО и сегментацию базы.

## Как присвоить теги

1. Перейдите в раздел **Исполнители** → вкладка **Теги исполнителей**.
2. Создайте новый тег или выберите существующий.
3. Загрузите файл со списком исполнителей, которым нужно присвоить тег.
4. Подтвердите операцию.

## Примеры использования

| Тег | Применение |
|---|---|
| `москва` | Исполнители из Москвы |
| `курьеры` | Исполнители-курьеры |
| `проект-А` | Исполнители на конкретном проекте |
| `декабрь-2025` | Исполнители определённого периода |

## Зачем нужны теги

- **ЭДО** — отправка документов группе исполнителей по тегу
- **Аналитика** — фильтрация и анализ по группам
- **Управление** — быстрый поиск и выборка нужных исполнителей
