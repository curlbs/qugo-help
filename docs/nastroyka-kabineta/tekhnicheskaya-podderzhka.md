# Обращение в техническую поддержку

Если у вас возникли вопросы по работе с платформой, обратитесь к нашим специалистам.

## Режим работы

| | Время (МСК) |
|---|---|
| Будни | 9:00 — 21:00 |
| Выходные | 10:00 — 19:00 |

## Как написать в поддержку

1. Войдите в [личный кабинет](https://lk.qugo.ru).
2. Нажмите на **иконку чата** в правом нижнем углу экрана — она доступна на любой странице.
3. Опишите вопрос и отправьте сообщение.

## С чем помогут специалисты

- Подключение функционала подписания актов с исполнителями
- Настройка работы с физлицами без специального налогового статуса
- Запуск выплат по номеру карты
- Любые вопросы по функциональности сервиса
