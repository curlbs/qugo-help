# Электронный документооборот с исполнителями

Платформа позволяет подписывать и отправлять документы исполнителям в электронном виде. Исполнители получают документы в мобильном приложении в разделе «ЭДО».

## Как настроить ЭДО

1. Войдите в [личный кабинет](https://lk.qugo.ru).
2. Перейдите в раздел **Документы** → **ЭДО**.
3. Загрузите документы, которые необходимо подписать или направить исполнителям.
4. Обратитесь в [техническую поддержку](tekhnicheskaya-podderzhka) для активации.

## Какие документы можно отправлять

- Договоры и дополнительные соглашения
- Акты выполненных работ
- Любые другие документы, требующие подписи

## Как исполнитель получает документы

Документы приходят в мобильное приложение Qugo в раздел **ЭДО**. Исполнитель может ознакомиться с ними и подписать электронной подписью.

:::tip Совет
Чтобы отправить документы группе исполнителей, используйте [массовое тегирование](../zadaniya-i-vyplaty/tegirovanie) — присвойте тег нужным исполнителям и направьте документы по тегу.
:::
