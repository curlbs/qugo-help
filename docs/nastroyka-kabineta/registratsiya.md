# Регистрация заказчика на платформе

Чтобы начать работу с Qugo, зарегистрируйте компанию на платформе. Процесс занимает несколько минут.

## Как зарегистрироваться

1. Откройте [qugo.ru](https://qugo.ru) и нажмите **Зарегистрироваться**.
2. Укажите email и придумайте пароль.
3. Подтвердите email по ссылке из письма.
4. Заполните данные компании: ИНН, название, контактное лицо.
5. Дождитесь проверки — обычно это занимает до 1 рабочего дня.

После прохождения проверки вам откроется полный доступ к личному кабинету.

:::tip Совет
Подготовьте заранее ИНН и реквизиты компании — это ускорит регистрацию.
:::
